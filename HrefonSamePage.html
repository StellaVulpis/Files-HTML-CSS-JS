Mod description

THIS MOD REQUIRES ALL SKYRIM DLCS, AND THE OTHER RACES MODS BELOW THE DESCRIPTION.(Links are attached.)

-Dawnguard

-Dragonborn

-RaceCompatibility by TMPhoenix

-Cellan Race and NPC by BadPup

 (Two esp files are in one same archive file.)

-Lykaios Race by Apalmemnom

-Fennec Fox Race by Siikfox
==========================================================================
I created this mod because I always wanted to see the beautiful races like Cellan, Fennec,
Lykaios on Tamriel with the other original races altogether.
It doesn't replace all the npcs, because it didn't seem necessary to me changing
all of 'em like the other mods do.
(It seemed more natural that there're racial diversity on tamriel to me, like original Skyrim.)
(Nord, Argonian, Khajiit, Breton, Imperial...etc)
I used the CK tool and SSE Edit to make this race replacer.
==========================================================================

Warning : If you used console command 'setrace' to npcs, this mod might not work properly.
If you've already modified npcs with commands in the game,
you might need to load older save that npcs weren't modified with console,
or start a new game to get this mod work properly.
SETRACE console command is very unstable, I recommend you not to use it.
